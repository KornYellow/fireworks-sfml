เป็นโปรเจคที่ทำเล่น ๆ ซึ่งวางแผนไว้ว่าจะทำตอนปีใหม่ไว้เป็นอวยพรวันปีใหม่ แต่ก็ทำไม่ทันฮ่า ๆ ในตัวโปรแกรมก็จะแสดงผลโดยใช้ SFML

# วิธีติดตั้งและรันโปรแกรม (Linux)

Dependencies ของผม :
- `SFML` 2.5

คำสั่งรัน
```console
./fireworks
```

คำสั่งคอมไพล์ (กรณีแก้ไขไฟล์)
```console
./compile.sh
```
